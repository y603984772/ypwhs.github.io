linux相关命令

发表于 2015-02-23 | 更新于 2016-12-25

apt-get

更新源 更新所有包 安装包 卸载包 卸载并删除配置文件

ls

列出文件 列出文件详细信息(权限,所有者等) 列出包括隐藏文件的所有文件

cat

查看文件内容

uname

列出系统信息 比如:Darwin YPW-MacBook-Pro.local 14.1.0 Darwin Kernel Version 14.1.0: Mon Dec 22 23:10:38 PST 2014; root:xnu-2782.10.72~2/RELEASE_X86_64 x86_64

mkdir

创建文件夹

创建一个目录 创建多层目录

wget

下载文件

效果如下:

tar

zcvf是压缩 xzvf是解压

mv,rm,cp

移动,删除,复制,重命名

pwd

显示当前目录

useradd,passad

useradd -g www-data -d /var/www ftpypw 添加用户 -g 用户组 -d 用户目录 用户名称 设置密码

cat

查看文件内容

chmod,chown

设置目录权限 设置目录所有者

du

export

添加环境变量

df

列出U盘或硬盘

mount

挂载U盘

mkfs

格式化U盘为fat32格式

screen

Ctrl+D退出这个窗口

ln

创建一个链接

a2enmod
